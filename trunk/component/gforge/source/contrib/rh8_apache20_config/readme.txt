Sample configuration files for RedHat 8.0/9.0
php 4.2.2/apache 2.0.40

This configuration files assume that:
- your installation of gforge is in /var/www/gforge. 
- your database name is gforge, username gforge, password gforge and the
  database server is localhost

MAKE BACKUP COPIES OF YOUR EXISTING FILES BEFORE COPYING THESE FILES!

the httpd.conf is  usually located in /etc/httpd/conf
the php.conf file is usually located in /etc/httpd/conf.d
the php.ini file is usually located in /etc
the local.inc file in /etc/gforge
