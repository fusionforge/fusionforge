Reglas de formato

_texto_
Hacer una énfasis
__texto__
Hacer una énfasis fuerte
@@texto@@
Hacer un pequeño código
''texto|idioma|código fuente''
Hacer una pequeña cita
>texto
Hacer un párrafo de cita
[texto|URI|idioma|título]
Hacer un enlace hacia una página, los parámetros son opcionales
((imagen|texto alternativo|alineación))
Agregar una imagen, alineación puede valer L (izquierda), D (derecha) o C (centro)
texto
Qualquier texto escrito simplemente será convertido en un párrafo.
!título1, !!título2, !!!título3, etc.
Crear un título de un nivel igual al nombre de !
-texto o *texto
Hacer una lista de balazos
#texto
Hacer una lista numerada
;término:definición
Hacer una lista de definiciones
??acrónimo|título??
Hacer un acrónimo
[ESPACIO]texto o {{{texto}}}
El texto será preformateado, útil para escribir pedazos de código fuente
==== o ---- (al menos 4)
Una línea de separación horizontal
\
Si usted no quiere que el formato wiki se aplique, ponga una diagonal invertida antes de los caracteres especiales.
